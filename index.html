Welcome

"What's all this, then?", you may ask. Well this is my site. It's where I, a lowely front-end developer, experiment with web technologies. So expect things to break from time to time (or always if you are using anything below IE10). It's built in Jekyll, uses HTML5, SCSS, jQuery, flexbox (in parts) and utilizes my own simple grid system. About half of these I get to use professionally, while the remainder are new to me.

This entire site is a work in progress and probalbly will be as long as I maintain it. Best viewed in Firefox or Chrome.
